# 📊 Amazon Sales Analysis Dashboard

## 📌 Project Overview
This project demonstrates how **data analytics** can transform raw Amazon sales data into actionable business insights.  
The dashboard provides a comprehensive view of **sales performance, customer behavior, and operational efficiency**, enabling smarter decision-making.

- **Total Revenue:** $48.3M  
- **Top Products:** T-shirts (>50% revenue), Shirts, Blazers  
- **Key Regions:** Maharashtra & Karnataka lead in order volumes  
- **Fulfillment:** 65%+ orders handled by Amazon logistics  

---

## 🚀 Features
- **Product & Category Analysis** → Identify high-performing items (T-shirts, Shirts) and low-performing categories (Shoes, Socks).  
- **Regional Insights** → Highlight demand concentration in states like Maharashtra & Karnataka.  
- **Customer Behavior** → Size preferences (M, L, XL dominate) for better inventory planning.  
- **Operational Analytics** → Fulfillment & courier status analysis showing strong logistics efficiency.  
- **Sales Trends** → Daily sales visualization to forecast seasonal demand and optimize inventory.  

---

## 🛠 Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, Matplotlib, Seaborn  
- **Visualization:** Interactive dashboards & charts  
- **Dataset:** Real transactional sales data (products, customers, regions, fulfillment)  

---

## 📈 Key Insights
- Revenue is **highly skewed** toward a few categories (Pareto principle).  
- **Customer satisfaction** is supported by efficient fulfillment (majority delivered successfully).  
- **Predictive analytics** helps forecast demand and plan inventory for peak seasons.  
- Regional focus on **Maharashtra & Karnataka** can drive targeted growth strategies.  

---

## 🎯 Mission & Vision
- **Mission:** Transform raw sales data into actionable insights for better decision-making.  
- **Vision:** Build a **data-driven retail system** leveraging predictive analytics, customer-centric strategies, and operational excellence.
